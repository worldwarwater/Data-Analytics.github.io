# BMI Calculator

## Overview
A Python-based Body Mass Index (BMI) calculator that accepts user input for weight (pounds) and height (feet and inches), calculates BMI using the standard formula, and classifies the result into WHO health categories.

## Features
- Accepts height in feet and inches format (e.g., `6.1` for 6 feet 1 inch)
- Converts imperial units for BMI calculation using the formula: `BMI = (weight in lbs * 703) / (height in inches)^2`
- Classifies results into 6 categories: Underweight, Normal weight, Overweight, Obese, Severely Obese, Morbidly Obese
- Includes input validation and error handling for invalid entries

## How to Run
```bash
jupyter notebook bmi_calculator.ipynb
```
