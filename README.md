## Projects

### Marketing Campaign Segmentation & ROI Analysis | Python, SQL, Scikit-learn, Looker Studio

Analyzed 2,240 customers to identify high-value segments using RFM analysis and K-Means clustering (K=4), then measured campaign ROI across web, catalog, and store channels. Applied Chi-Square, ANOVA, T-Test, and Mann-Whitney U tests to validate segment differences. Produced 30+ visualizations and data-driven budget reallocation recommendations.

**Key Finding:** Champions and Loyal segments drove the majority of revenue, with store purchases delivering the highest ROI (1,163%) despite the lowest acquisition cost.

### Job Market Analysis Pipeline | Python, SQL, SQLite, Looker Studio

Built an end-to-end data pipeline analyzing 2,500 data role job postings to uncover salary trends, in-demand skills, and the remote work pay gap. Designed a normalized SQLite database, wrote 10 SQL analysis queries, and validated findings with statistical tests (T-tests, ANOVA, Chi-square, Cohen's d). Generated 8 publication-quality visualizations.

**Key Finding:** ML/AI skills (TensorFlow, Spark MLlib) command $20-30K salary premiums, while remote roles pay ~$4,400 more on average but attract 59% more applicants.

[View Code](job-market-analysis/) | [View Dashboard](https://lookerstudio.google.com/s/g3VbO0T8B4c)

### Customer Churn Analysis & Prediction | Python, Scikit-learn, Looker Studio

Analyzed 7,043 telecom customers to identify churn drivers and built a logistic regression model (ROC-AUC ~0.74) to predict at-risk customers. Engineered features from contract type, tenure, payment method, and service bundles. Created an interactive Looker Studio dashboard for stakeholder exploration.

**Key Finding:** Month-to-month contracts churn at 42% — converting just 10% to annual contracts could save ~$14K/month in revenue.

[View Code](customer-churn-analysis/) | [View Dashboard](https://lookerstudio.google.com/s/ra8vaWVRUzQ)
